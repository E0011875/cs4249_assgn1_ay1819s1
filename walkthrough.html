Interface Walkthrough

Dear participants, please read and understand the interface before the experiment.

Upon starting the experiment, you will be greeted by an interface that looks like this on the top of your screen:

The information provided by the interface are as follows:

  • Trial: Current progress / total no. of trials
  • Menu Type: The type of menu you will be using
  • Device: Mouse pointing device needed to perform a selection on the menu. To help you readily switch between the two pointing devices, each device name is highlighted in a different colour. Mouse is highlighted in Green while Trackpad is in Yellow
  • Task: The number of items in a selection. single vs dual.
  • Menu Depth: Number of menu layer included in a trial
  • Please select: Target menu item
  • Item Selected: Selected menu item. No item is selected by default.
  • Next: Proceed to the next trial once done. If no selection is made, a dialog box will appear to prompt you to select before advancing to the next trial.

The followings are the commands for each type of menu introduced in this experiment:

Marking Menu

Actions Commands
Open Menu Hold down left mouse button
Select Hold down left mouse button and stroke towards the target menu item. Release mouse down once done.
Reset Release mouse down

*Expert users can make a fast stroke to select target item and need not use the menu popup invoked by holding the left mouse button.

*Should there be more choices available for a particular selection, stroking towards it will open up a new menu.

Radial Menu

Actions Commands
Open Menu Right Click
Select Left Click
Reset Right Click

*Should there be more choices available for a particular selection, clicking on it will open up a new menu.

Note

A practice session has been prepared for you before the actual experiment. The practice session is not timed, nor will any data be collected for this session. Please take your time to attempt the course in order to maximize your familiarity with both menus. Read the instructions carefully on each screen and select the correct items on the presented menu accordingly. You will be prompted to switch between pointing devices from time to time.

Once you are done with the practice session, you can proceed with the actual experiment. For every trial conducted in the actual experiment, both the time and the accuracy of the items selected on the menu will be recorded down.

Please take as long as you need to read the Instructions. Approach the experiment coordinator should you have any questions.

Once ready, pressed “Begin Practise Trials” to begin practising, otherwise pressed “Begin Main Experiment” to proceed with the actual experiment.
